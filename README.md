# Search Operators
| Operator         | Description                       | Syntax                               | Example                           |
| ---------------- | --------------------------------- | ------------------------------------ | --------------------------------- |
| !       | Matches the form of the word (case, number, and tense)                               | !&lt;word>   | buy !apples wholesale               |
| +       | Matches stop words (auxiliary parts of speech, pronouns, and any words that do not add meaning on their own). | +&lt;word>   | work +from home  |
| " "     | Matches the exact number of words. Ads are triggered by search queries that contain the keyword without any additional words. | "&lt;keywords>" | "buy cars" |
| [ ]     | Matches the order of words. This operator will match any grammatical forms and stop words found inside the brackets. | [&lt;word>] | tickets [london paris] |
| () and &#124; | Groups words in complex search queries. | (&lt;word>&#124;&lt;word>) | buy (coat&#124;jacket) | 
| $ anchor() | Will look for web pages that have the term specified in () in the anchor text of links. | $ anchor(&lt;word>) | $ anchor(seo) |
| date:   | Lets you filter web pages by their last modified date. You must specify the year while the day or month may be replaced with *. | date:&lt;YYYMMDD> | seo date:20200815 |
| domain: | Will look for the web pages of the domain specified. This works with domains ending in “com”. | domain:&lt;word> | seo domain:seosly |
| host:   | Will search for web pages that are hosted on the host specified. It is identical to the url: search operator if you only specify the hostname. | host:www.second-level-domain.top-level-domain | seo host:www.seosly.com |
| inurl:  | Will look for web pages containing the word or words specified in their URL address. | inurl:&lt;word> | inurl:seo |
| lang:   | Will look for web pages that are in the language specified. The languages supported by Yandex include Russian (ru), Ukrainian (uk), Belarusian (be), English (en), French (fr), German (de), Kazakh (kk), Tatar (tt), Turkish (tr). | &lt;Alpha-2 code> | seo lang:en | 
| mime:   | Will look for specific file types in search results based on the keyword specified. | mime:&lt;file types> | seo mime:pdf |
| rhost:  | Works the same as host: except that with rhost: the hostname must be written in the reverse order (starting from the top-level domain). | rhost:top-level-domain.second-level-domain.www | seo rhost:com.seosly.www |
| site:   | Will look for the web pages and subdomains of the website specified. | site:&lt;website> | site:seosly.com |
| title:  | Will look for web pages with the keyword specified in the title. | title:&lt;keywords> | title:seo audit |
| url:    | Will look for web pages with a specific URL address. You should put the URL in quotation marks if it contains characters, such as ‘ " ( ) _. | url:&lt;website> | url:seosly.com |

From [Seosly.com](https://seosly.com/blog/yandex-search-operators/#url)
